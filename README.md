# ⚖️ Infotech Law Chatbot (Asisten Hukum Digital)

Infotech Law Chatbot adalah sebuah aplikasi asisten pintar berbasis AI yang dirancang khusus untuk memberikan edukasi, penjelasan istilah, serta solusi awal terkait permasalahan **Hukum Teknologi Informasi (Cyber Law)** dan regulasi digital di Indonesia. 

Proyek ini dibangun untuk memenuhi **Final Project - LLM-Based Tools and Gemini API Integration for Data Scientists**.

---

## 🚀 Fitur Utama & Parameter Kreatif
*   **Persona Khusus Hukum Digital:** Menggunakan *System Instruction* agar AI bertindak sebagai pakar hukum IT di Indonesia (fokus pada UU ITE, Perlindungan Data Pribadi, dan Hukum Administrasi Negara terkait digitalisasi).
*   **Antarmuka Interaktif:** Dibangun menggunakan framework **Streamlit** untuk UI chat yang bersih, modern, dan mudah digunakan.
*   **Memori Percakapan (Session State):** Bot mampu mengingat riwayat obrolan dalam satu sesi percakapan agar komunikasi berjalan dua arah secara kontekstual.

---

## 🛠️ Stack Teknologi
*   **Bahasa Pemrograman:** Python 3
*   **LLM Engine:** Google Gemini API (`models/gemini-1.5-flash`)
*   **Framework UI:** Streamlit
*   **Pengelola Environment:** `python-dotenv` & Virtual Environment (`venv`)
